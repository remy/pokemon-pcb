PCB Net Path Editor

Mouse wheel zoom. Hold space and drag to pan. Middle mouse also pans.
Zoom: 1.00x
Tool
Polygon mode: press and drag to lasso a shape. Release to create the net path.
Magnet mode: drag to trace; points snap to nearby high-contrast PCB features.
Select mode: drag anchors to reshape. Right-click path/anchor to add or delete nodes.
Click an anchor to select it, then press B to toggle that node between corner/smooth.
0.18
Only the selected path is rendered in the canvas.
Shortcuts: [ previous path, ] next path.
Undo: Ctrl/Cmd+Z (last 100 edits, per side).
